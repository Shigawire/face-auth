Einstellungen

Hier kannst du Einstellungen für die Microsoft Cognitive Services API bearbeiten. Es muss ein gültiger API-Key und eine PersonGroup angelegt werden.

Cognitive API Key

 {{vm.settings.apiKey}}

API Person Groups

 {{vm.settings.personGroup}}

Achtung!

Sobald du die personGroup änderst, werden alle Nutzer in der App gelöscht.

Da die Microsoft Cognitive Services API immer nur eine personGroup durchsucht, können Gesichter in anderen Gruppen nicht mehr identifiziert werden.

Bitte warte ein paar Minuten nach dem Anlegen einer personGroup. Andernfalls ist bei der ersten Registrierung der Trainingsmodus noch nicht beendet. Währenddessen können keine weiteren Gesichter angelegt werden - die API wird unerwartete Ergebnisse zurückliefern.
